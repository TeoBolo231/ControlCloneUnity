How to set up the system:
- Set up the Unity Third Person Asset
- Attach the script called TK_Controller_2 to the third person character
- Drop the "Canvas" present inside the UI folder into the game scene
- Go to the character Inspector, open "Player Input" and drag the "InputTK" action map in the "Actions" field.
- In the character Inspector, open "TK_Controller_2" and fill the "Feed Me!" section 
- Assign Layers and Tags based on the needs
- Press "Play"

NOTE: All the Prefabs that you want to instantiate runtime need to be saved in the "Resources" folder.

Controls
- E: Grab/Drop Object
- Q: Shield
- Left Click: Shoot Object/Shield
- C: Levitate
